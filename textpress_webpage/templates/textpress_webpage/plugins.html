Uploading Plugins

If you want to upload your plugin and register it in the TextPress plugin repository you have to have a developer login. If don't have one you can get one in a few seconds. After creating your developer id you can start uploading plugins from the textpress-management.py utility.

Just make sure that your plugin is associated with a TextPress instance and execute the following command from within a terminal: 

python textpress-management.py <path to instance> upload_plugin <plugin name>

The management utility will ask you for your credentials and upload and register the plugin then. If you upload a plugin that is already registered it checks if your developer id is the same. If it is it either replaces the current version or registeres a new one. If a new version is uploaded it's also announced on the index page.
