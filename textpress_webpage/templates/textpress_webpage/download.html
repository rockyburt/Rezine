Download TextPress

Interested in TextPress? Wait a minute! Just to clearify something: TextPress is still in the alpha stage and it's not the best idea to use it on public hosts. The only way to get the source code by now is checking it out from the mercurial repository and it could be that temporary debugging code found its way into the source tree for some checkins.

If you really plan to use TextPress for your webpage head over to the #textpress channel on freenode and ask for a stable hg node which you can check out.

Getting The Source

You need to have mercurial installed until the beta version is released. After installing mercurial you can execute the following command in a terminal to download the most recent version of mercurial: 

$ hg clone http://hg.pocoo.org/textpress-main textpress

Afterwards you can install the package by executing this command: 

$ python setup.py install

Setting up a Development Instance

For development and debugging purposes you have to setup a TextPress instance. You have to do the same for public blogs too but this process will become simpler once the beta is out. So far you have to use the terminal to get TextPress running: 

$ mkdir my_instance
$ textpress-management.py my_instance serve

If you plan to develop plugins it's a good idea to enable the automatic code reloader with the -r parameter and the debugging system with the -d parameter.

The first time you start up textpress you will see an installation wizzard. Just follow the steps and you're done.

Installing Plugins

TextPress plugins come as .plugin files. Of course you can use a normal zip compatible extractor but TextPress can do that on its own too. First you have to download the plugin to your computer, afterwards log into the TextPress admin panel and go to the "Options -> Plugins" menu. There you can then upload the plugin. The plugin is then installed into the instance folder where you can also view the source code if you like.

Creating Plugin Packages

If you want to create plugin packages to distribute plugins you can do so by using the textpress-management utiltity: 

$ textpress-management.py dump_plugin plugin_name

This will copy the dumped plugin into your current working directory with the proposed filename. You can also replace the dump_plugin parameter with upload_plugin to push the plugin to the TextPress server and register it in the plugin repository. The latter however requires a developer login.
