Download TextPress

Got interested? Then you can download TextPress from below:

  • TextPress 0.1, regular
  • TextPress 0.1, quickstart

The regular distribution contains the TextPress source code, the quickstart distribution also contains some dependencies so that you can get started quickly on a plugin webhost.

Unsure which distribution to chose? You can check which one is better for you on the requirements page.

Both have exactly the same features, the only difference is the way you install and upgrade them.
