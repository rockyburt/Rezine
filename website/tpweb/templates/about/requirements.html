Requirements

Unlike many other blog engines, TextPress is not written in PHP but in the general purpose language Python. Unfortunately many cheap hosts do not provide Python hosting so is currently a little bit tricky to get TextPress running on cheap webhosts.

You have to have at least Python 2.4 and the following Python modules installed. If you are using easy_install to install TextPress these are installed automatically for you.

  • SQLAlchemy 0.4 or higher
  • Jinja 1.2 or higher
  • Werkzeug 0.2 or higher

Additionally one of the database systems supported by SQLAlchemy must be available. Currently these are MySQL, sqlite, postgresql, MSSQL, and firebird.

The following webservers and interfaces are supported by TextPress:

  • Apache, via FastCGI, SCGI, AJP, CGI, mod_python, or mod_wsgi
  • lighttpd, via FastCGI, SCGI, or CGI
  • nginx, via FastCGI and CGI
