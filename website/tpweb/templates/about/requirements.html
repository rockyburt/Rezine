Requirements

Unlike many other blog engines, TextPress is not written in PHP but in the general purpose language Python. Unfortunately many cheap hosts do not provide Python hosting so is currently a little bit tricky to get TextPress running on cheap webhosts.

There are two TextPress “distributions”. One is the regular distribution which you should chose if you want to maintain multiple TextPress installations and if you have shell access to the server. It just includes the TextPress sourcecode and a script to install the application code in the system wide python extension folder. The other distribution is the “quickstart” distribution. It comes with many of the depdenceny modules and should work more or less out of the box in a CGI, FastCGI, mod_python or mod_wsgi environment.

The table below lists the minimal requirements for TextPress:

regular quickstart
minimal Python version 2.4 2.4
SQLAlchemy 0.4 or higher yes no
Jinja 1.2 or higher yes no
Werkzeug 0.2 or higher yes no

Additionally one of the database systems supported by SQLAlchemy must be available.

The following webservers and interfaces are supported by TextPress:

  • Apache, via FastCGI, SCGI, AJP, CGI, mod_python, or mod_wsgi
  • lighttpd, via FastCGI, SCGI, or CGI
  • nginx, via FastCGI and CGI
