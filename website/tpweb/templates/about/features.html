Features

  • Valid HTML4. While it looked like XHTML would become the standard for long time, HTML is getting a lot of attention lately again. The reason for this is the current bad support of XHTML in browsers and the huge number of pages with an XHTML doctype but HTML4 contents. TextPress currently sticks to HTML4 and we will make the switch to either HTML5 / XHTML5 of XHTML2, depending on what will best supported at that time.
  • Atom support. TextPress supports the most recent Atom standard for best interoperability with other applications and web services.
  • Extensible. TextPress is build on a solid core that is extensible by plugins. These plugins can extend nearly every part of the application.
  • Pingback support. TextPress comes with full support for pingback for cross-blog communication.
  • Mass-Hosting ready. TextPress does not support multiple blogs in one blog but is developed in a way that you can easily host dozens of blog and upgrade them all in one go. Adding new blogs once you've got TextPress running is a matter of a few seconds.
  • Simple Web Setup. The initial web setups helps you getting TextPress running in less than two minutes.
  • Unicode Ready. TextPress uses unicode internally everywhere. This allows you to write posts in any language and also mix different languages in the same post.
  • Simplified HTML. As blog author you often want to write the HTML for your blog posts on your own but want to have automatic paragraphs or some other helpers (such as automatically escaped text inside <pre>-tags). TextPress comes with an intelligent algorithm that ensures your code is valid HTML on the one hand, and easy to write on the other.
