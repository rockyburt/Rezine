Our Inspiration

TextPress was created out of the desire to have an extensible blogging platform in an agile language that was secure and user friendly at the same time.

The main inspirations for TextPress are obviously WordPress and TextPattern which both have great user interfaces and are extensible in a similar way TextPress is.

Most if the ideas in terms of implementation and code style were derived from the Django web framework and trac.
