Extend TextPress

TextPress was designed to be extensible via plugins. A lot of the functionality provided by TextPress is already implemented as plugin like for example file uploads or the themes.

If you want to create your own plugins or themes have a look at the developer documentation.

In the near future we will add a central plugin repository under this URL. Until then have a look at the TextPress wiki.
