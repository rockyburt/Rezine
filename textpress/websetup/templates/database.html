Database

TextPress uses a database to store blog posts and other information. The only information it requires therefore is a database URI. Depending on your database this URI looks different. Have a look at the following table for examples of database URIs.

SQLite
sqlite:///database.db — connect to a database that is located at a specific absolute path on the file system of the server. Note that on Mac OS X and Linux systems three slashes mean relative to the instance folder and four slashes represent an absolute path. On Windows you have to use three slashes, a drive letter and the full path.
MySQL
mysql://username:password@host/database_name
Postgres
postgres://username:password@host/database_name
